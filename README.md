# WebCrawler

A web crawler, search engine, and link graph built in Python with a C# API layer.

## What it does
- Crawls websites and extracts titles and links
- Stores crawled pages in SQLite
- Searches stored pages by title
- Maps relationships between sites

## Stack
- Python (crawler, parser, storage, search, graph)
- C# ASP.NET Core (API layer — coming later)
